Designe McDonalds

Desenvolvi um projeto de design digital para o McDonald’s com base na identidade visual da marca, mas sem copiar o site oficial. Criei um layout próprio usando HTML, CSS e JavaScript, focando em uma navegação simples, cores características (vermelho e amarelo), e elementos visuais personalizados que remetem aos produtos da marca. O design é responsivo e inclui animações leves para melhorar a experiência do usuário. Todo o projeto foi feito do zero, como uma proposta criativa e independente, respeitando a essência da marca com um toque original.

International bank design

Desenvolvi um projeto de design digital para um banco internacional, criando uma interface moderna e funcional que reflete a imagem de segurança e confiança da instituição. Utilizei HTML, CSS e JavaScript para criar um layout intuitivo e responsivo, com foco em uma navegação simples e acessível. A paleta de cores foi escolhida para transmitir profissionalismo, com tons de azul e cinza predominando. Além disso, adicionei animações sutis e elementos gráficos personalizados, garantindo que a experiência do usuário fosse fluida e agradável. O design foi feito do zero, sem se basear em nenhum modelo existente, respeitando a identidade visual do banco enquanto trouxe uma abordagem inovadora e eficiente para os serviços digitais.

Pet-shop design

Desenvolvi um projeto de design digital para um pet shop, com o objetivo de criar uma experiência agradável e intuitiva para os donos de animais. Utilizando HTML, CSS e JavaScript, criei uma interface colorida e amigável, com um layout focado na navegação fácil e visual atraente para os visitantes. A paleta de cores foi inspirada em tons vibrantes e naturais, como verde, laranja e azul, que remetem ao universo dos pets. Além disso, adicionei animações divertidas e interativas, como destaque em produtos e categorias, para tornar a experiência mais dinâmica. O design é responsivo e foi criado do zero, respeitando a identidade do pet shop, ao mesmo tempo em que traz uma abordagem inovadora para facilitar a compra de produtos e serviços.
